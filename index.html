Petfood

Alimentação de pets

O Petfood foi um projeto desenvolvido como um trabalho no COLTEC - UFMG, que consistia em criar um aplicativo que controlasse um mecanismo para alimentação de cachorros, gatos, pets em geral. Fui responsável pela parte frontend do aplicativo, feito com java no Android Studio, e usando a integração como banco de dados pelo Firebase.

Enduro - Coltec

Site para o Enduro a pé - Coltec

O Enduro - Coltec foi um projeto desenvolvido durante meu estágio no COLTEC - UFMG, onde fui responsável pela criação do site oficial do evento Enduro, que é uma atividade esportiva de competição que consiste em cumprir trechos de caminhada (trekking), mantendo a regularidade (distância/tempo) estabelecida na planilha de orientação. Reservas Coltec

Sistema para gerenciamento de reservas do colégio

O Sistema de Reservas é um sistema agora utilizado por todo o coltec, que permite com que alunos e professores possam reservar algum espaço do colégio, seja uma quadra, auditório, salas, laboratórios, etc. Fazendo com que eliminamos o uso de papel e facilite a vida de todos que precisam. Foi desenvolvido para um trabalho de Desenvolvimento de Aplicações Web (DAW) pelo meu grupo.
